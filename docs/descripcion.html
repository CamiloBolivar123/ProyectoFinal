Proyecto Acueducto en Benjamín Herrera, Guaduas

Descripción del Proyecto

Este proyecto tiene como objetivo abordar un problema recurrente de suministro de agua potable que afecta al barrio Benjamín Herrera en Guaduas, Cundinamarca, Colombia. Durante las horas de la noche, los residentes enfrentan cortes frecuentes en el servicio de acueducto, lo que genera inconvenientes en su vida diaria. Este proyecto busca investigar las causas subyacentes de este problema y proponer soluciones viables para mejorar el servicio de agua.

Tecnologías Utilizadas

Este proyecto hace uso de varias tecnologías web para su desarrollo:

  • HTML: Se utiliza para estructurar las páginas web, creando la base de la interfaz de usuario.
  • CSS: Se emplea para dar estilo y formato a la página, mejorando la apariencia visual.
  • JavaScript: Permite agregar interactividad a la página, como la visualización de mapas interactivos.
  • Leaflet.js: Es una librería JavaScript para crear mapas interactivos. En este proyecto, se utiliza para mostrar la zona afectada por el problema del acueducto.
  • Git y GitHub: Git se usa para el control de versiones del proyecto y GitHub para almacenar y gestionar el código fuente de forma remota.
